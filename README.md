# 八大行星轨道舱

这是“八大行星轨道舱”的 GitHub Pages 静态发布目录，面向 H5 科普互动作品提交。

## 在线入口

固定静态入口：

```text
https://flooder55.github.io/planet-orbit-gate/
```

GitHub Pages 入口文件为：

```text
index.html
```

## 目录说明

- `index.html`：H5 科普互动页面。
- `assets/`：Three.js 依赖、NASA 行星 GLB 模型和行星图片资源。
- `.nojekyll`：确保 GitHub Pages 按静态资源原样发布。

## 注意

此目录用于 GitHub Pages 在线浏览版本。页面包含 3D 星系入口、八大行星轨道、NASA 模型浏览、高清图谱、竖式浏览和手动星球工坊。手机端轨道页的行星资料默认收起为底部小条，避免遮挡星轨展示。
